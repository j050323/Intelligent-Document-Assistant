## 快速开始

智能文档助手是一个功能强大的文档管理系统，帮助您轻松管理和组织各类文档。

### 主要功能

- 📤 **文档上传**: 支持 PDF、Word、TXT 格式，单个文件最大 100MB
- 📁 **文件夹管理**: 创建多层级文件夹，有序组织文档
- 🔍 **智能搜索**: 快速查找文档，支持文件名搜索和类型筛选
- 👁️ **在线预览**: 无需下载即可预览文档内容
- 📥 **批量操作**: 批量上传、批量删除，提高工作效率
- 💾 **存储管理**: 实时查看存储空间使用情况

### 系统要求

- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **网络**: 稳定的互联网连接
- **文件格式**: PDF (.pdf), Word (.doc, .docx), TXT (.txt)

## 用户注册与登录

### 注册新账户

1. 访问系统首页
2. 点击"注册"按钮
3. 填写注册信息：
   - 用户名（3-20个字符）
   - 邮箱地址
   - 密码（至少8个字符，包含大小写字母和数字）
4. 点击"注册"按钮
5. 查收验证邮件并点击验证链接
6. 验证成功后即可登录

**注意事项**:
- 邮箱地址必须真实有效，用于接收验证邮件
- 密码需要包含大小写字母和数字，确保账户安全
- 如果未收到验证邮件，请检查垃圾邮件文件夹

### 登录系统

1. 访问系统首页
2. 输入邮箱和密码
3. 点击"登录"按钮

**忘记密码？**
1. 点击登录页面的"忘记密码"链接
2. 输入注册邮箱
3. 查收密码重置邮件
4. 点击邮件中的重置链接
5. 设置新密码

## 文档管理

### 上传文档

#### 单个文档上传

1. 登录系统后，进入"文档管理"页面
2. 点击"上传文档"按钮
3. 选择要上传的文件
4. （可选）选择目标文件夹
5. 点击"确认上传"

**支持的格式**:
- PDF 文档 (.pdf)
- Word 文档 (.doc, .docx)
- 文本文件 (.txt)

**文件大小限制**: 单个文件最大 100MB

#### 批量上传

1. 点击"批量上传"按钮
2. 选择多个文件（可按住 Ctrl/Cmd 键多选）
3. （可选）选择目标文件夹
4. 点击"开始上传"
5. 查看上传进度和结果

**提示**:
- 批量上传时，如果某个文件失败，其他文件会继续上传
- 上传完成后会显示详细的成功和失败列表

### 查看文档列表

#### 浏览文档

文档列表默认按上传时间倒序排列，显示以下信息：
- 文件名
- 文件类型（图标）
- 文件大小
- 所属文件夹
- 上传时间

#### 搜索文档

1. 在搜索框中输入关键词
2. 系统会实时搜索文件名包含关键词的文档
3. 点击搜索结果即可查看文档详情

#### 筛选文档

**按文件类型筛选**:
- 点击"文件类型"下拉菜单
- 选择 PDF、Word 或 TXT
- 列表会只显示选中类型的文档

**按文件夹筛选**:
- 在左侧文件夹树中点击文件夹
- 列表会只显示该文件夹中的文档

#### 排序文档

点击列表表头可以按以下字段排序：
- 文件名（A-Z 或 Z-A）
- 文件大小（从小到大或从大到小）
- 上传时间（最新或最旧）

### 预览文档

1. 在文档列表中找到要预览的文档
2. 点击文档名称或"预览"按钮
3. 系统会在新窗口中打开文档预览

**预览方式**:
- **PDF**: 直接在浏览器中显示
- **TXT**: 显示文本内容
- **Word**: 转换为可预览格式或提供下载

**提示**: 预览功能无需下载文件，节省时间和存储空间

### 下载文档

1. 在文档列表中找到要下载的文档
2. 点击"下载"按钮或右键菜单中的"下载"
3. 选择保存位置
4. 等待下载完成

**支持断点续传**: 如果下载中断，可以继续下载而不需要重新开始

### 重命名文档

1. 在文档列表中找到要重命名的文档
2. 点击"更多操作"菜单（三个点图标）
3. 选择"重命名"
4. 输入新的文件名
5. 点击"确认"

**注意**: 重命名只会修改文件名，不会影响文件内容

### 移动文档

1. 在文档列表中找到要移动的文档
2. 点击"更多操作"菜单
3. 选择"移动到"
4. 在弹出的文件夹树中选择目标文件夹
5. 点击"确认"

**提示**: 可以将文档移动到根目录或任何子文件夹

### 删除文档

#### 删除单个文档

1. 在文档列表中找到要删除的文档
2. 点击"更多操作"菜单
3. 选择"删除"
4. 在确认对话框中点击"确认删除"

#### 批量删除

1. 勾选要删除的多个文档
2. 点击"批量删除"按钮
3. 在确认对话框中点击"确认删除"

**警告**: 删除操作不可恢复，请谨慎操作！

**删除后**:
- 文档会从数据库和文件系统中永久删除
- 存储空间会相应释放
- 删除操作会记录在系统日志中

## 文件夹管理

### 创建文件夹

#### 创建根文件夹

1. 在文档管理页面，点击"新建文件夹"按钮
2. 输入文件夹名称
3. 点击"创建"

#### 创建子文件夹

1. 在左侧文件夹树中，右键点击父文件夹
2. 选择"新建子文件夹"
3. 输入文件夹名称
4. 点击"创建"

**文件夹命名规则**:
- 长度：1-255 个字符
- 不能包含特殊字符：`/ \ : * ? " < > |`
- 同一层级下不能有重名文件夹

### 浏览文件夹

**文件夹树视图**:
- 左侧显示文件夹树形结构
- 点击文件夹名称展开/折叠子文件夹
- 点击文件夹可查看其中的文档

**面包屑导航**:
- 顶部显示当前位置的完整路径
- 点击路径中的任意文件夹可快速跳转

### 重命名文件夹

1. 在文件夹树中找到要重命名的文件夹
2. 右键点击文件夹
3. 选择"重命名"
4. 输入新的文件夹名称
5. 点击"确认"

**注意**: 重命名文件夹不会影响其中的文档和子文件夹

### 删除文件夹

1. 在文件夹树中找到要删除的文件夹
2. 右键点击文件夹
3. 选择"删除"
4. 在确认对话框中点击"确认删除"

**重要提示**:
- 只能删除空文件夹
- 如果文件夹中包含文档或子文件夹，系统会提示先清空文件夹
- 删除操作不可恢复

### 文件夹层级结构

系统支持多层级文件夹结构，例如：

```
📁 我的文档
├── 📁 工作文档
│   ├── 📁 项目A
│   │   ├── 📄 需求文档.pdf
│   │   └── 📄 设计方案.docx
│   └── 📁 项目B
│       └── 📄 进度报告.pdf
├── 📁 个人文档
│   ├── 📄 简历.pdf
│   └── 📄 证书.pdf
└── 📁 学习资料
    ├── 📄 教程.pdf
    └── 📄 笔记.txt
```

**最佳实践**:
- 根据文档类型或项目创建顶层文件夹
- 使用子文件夹进一步细分
- 保持文件夹结构简洁清晰
- 定期整理和归档文档

## 存储空间管理

### 查看存储空间

在文档管理页面右上角，可以看到存储空间使用情况：

```
已使用: 256 MB / 1 GB (25.6%)
剩余: 768 MB
```

**显示信息**:
- 已使用空间
- 总配额
- 使用百分比
- 剩余空间

### 存储配额

每个用户都有存储配额限制，默认为 1GB。

**配额限制**:
- 上传文档时会检查配额
- 如果空间不足，上传会被拒绝
- 删除文档会释放空间

**接近配额警告**:
- 当使用量超过 90% 时，系统会显示警告
- 建议及时清理不需要的文档

### 释放存储空间

**方法 1: 删除不需要的文档**
1. 查看文档列表
2. 找到不再需要的文档
3. 删除这些文档
4. 存储空间会立即释放

**方法 2: 下载后删除**
1. 下载重要文档到本地
2. 从系统中删除这些文档
3. 释放云端存储空间

**方法 3: 压缩文档**
- 将多个小文档合并为一个文档
- 使用压缩工具减小文件大小
- 重新上传压缩后的文档

### 增加存储配额

如果需要更多存储空间，请联系管理员：

1. 点击右上角的用户菜单
2. 选择"联系管理员"
3. 说明需要增加配额的原因
4. 等待管理员审批

## 常见问题

### 上传相关

**Q: 为什么我的文件上传失败？**

A: 可能的原因：
1. 文件格式不支持（只支持 PDF、Word、TXT）
2. 文件大小超过 100MB
3. 存储空间不足
4. 网络连接不稳定

**解决方法**:
- 检查文件格式和大小
- 查看存储空间使用情况
- 尝试重新上传
- 联系管理员

**Q: 批量上传时部分文件失败怎么办？**

A: 系统会显示详细的失败原因。您可以：
1. 查看失败文件的错误信息
2. 修正问题后重新上传失败的文件
3. 成功上传的文件不需要重新上传

**Q: 上传速度很慢怎么办？**

A: 建议：
1. 检查网络连接
2. 避免在网络高峰期上传
3. 分批上传大文件
4. 使用有线网络而非 WiFi

### 文档管理相关

**Q: 如何快速找到某个文档？**

A: 使用搜索和筛选功能：
1. 在搜索框输入文件名关键词
2. 使用文件类型筛选
3. 使用文件夹筛选
4. 按上传时间排序

**Q: 可以恢复已删除的文档吗？**

A: 不可以。删除操作是永久性的，无法恢复。建议：
1. 删除前仔细确认
2. 重要文档先下载备份
3. 定期备份重要文档到本地

**Q: 为什么无法预览某些文档？**

A: 可能的原因：
1. 文档文件损坏
2. 文档格式特殊
3. 浏览器不支持

**解决方法**:
- 尝试下载后用本地软件打开
- 重新上传文档
- 更换浏览器

### 文件夹相关

**Q: 为什么无法删除文件夹？**

A: 只能删除空文件夹。如果文件夹中包含文档或子文件夹，需要先：
1. 删除或移动文件夹中的所有文档
2. 删除所有子文件夹
3. 然后才能删除该文件夹

**Q: 文件夹层级有限制吗？**

A: 理论上没有层级限制，但建议：
- 不超过 5 层
- 保持结构简洁
- 避免过深的嵌套

### 存储空间相关

**Q: 如何查看我使用了多少存储空间？**

A: 在文档管理页面右上角可以看到实时的存储空间使用情况。

**Q: 删除文档后存储空间没有立即释放？**

A: 存储空间应该立即释放。如果没有，请：
1. 刷新页面
2. 重新登录
3. 联系管理员

**Q: 如何申请更多存储空间？**

A: 联系系统管理员，说明需要增加配额的原因。

### 账户相关

**Q: 如何修改密码？**

A: 
1. 点击右上角的用户头像
2. 选择"个人设置"
3. 点击"修改密码"
4. 输入当前密码和新密码
5. 点击"保存"

**Q: 如何修改邮箱？**

A:
1. 进入"个人设置"
2. 点击"修改邮箱"
3. 输入新邮箱地址
4. 验证新邮箱
5. 完成修改

**Q: 如何上传头像？**

A:
1. 进入"个人设置"
2. 点击头像区域
3. 选择图片文件
4. 调整裁剪区域
5. 保存

### 安全相关

**Q: 我的文档安全吗？**

A: 系统采用多重安全措施：
1. JWT 令牌认证
2. 用户数据隔离
3. 权限验证
4. 操作日志记录
5. HTTPS 加密传输（生产环境）

**Q: 其他用户能看到我的文档吗？**

A: 不能。每个用户只能访问自己的文档。管理员可以访问所有文档，但所有访问都会记录在日志中。

**Q: 如何保护我的账户安全？**

A: 建议：
1. 使用强密码
2. 定期修改密码
3. 不要分享账户信息
4. 使用后及时退出
5. 不在公共电脑上保存密码

### 技术支持

**Q: 遇到问题如何获取帮助？**

A: 
1. 查看本使用指南
2. 查看常见问题解答
3. 联系系统管理员
4. 发送邮件到 support@docassistant.com

**Q: 如何报告 Bug？**

A:
1. 记录问题发生的步骤
2. 截图错误信息
3. 联系管理员或发送邮件
4. 提供详细的问题描述

## 使用技巧

### 高效管理文档

1. **合理规划文件夹结构**
   - 按项目、类型或时间分类
   - 保持结构简洁清晰
   - 定期整理归档

2. **使用描述性文件名**
   - 包含关键信息
   - 使用日期前缀（如：2023-12-16_报告.pdf）
   - 避免使用特殊字符

3. **定期清理**
   - 删除过期文档
   - 归档历史文档
   - 保持存储空间充足

4. **善用搜索功能**
   - 记住文档关键词
   - 使用筛选缩小范围
   - 利用排序快速定位

### 批量操作技巧

1. **批量上传**
   - 先整理好文件
   - 按文件夹分批上传
   - 检查上传结果

2. **批量删除**
   - 使用筛选功能
   - 勾选要删除的文档
   - 确认后一次性删除

### 存储空间优化

1. **压缩文档**
   - 使用 PDF 压缩工具
   - 降低图片质量
   - 删除不必要的页面

2. **定期备份**
   - 下载重要文档到本地
   - 使用外部存储备份
   - 删除云端副本

3. **合理使用配额**
   - 只上传必要的文档
   - 及时删除临时文件
   - 监控存储使用情况
